# Installation

To install on mac

```bash
brew tap timtylin/scholdoc
brew update
brew install scholdoc scholdoc-citeproc
```
To generate the document:
```bash
scholdoc hello.scholmd
```

[Guide](http://scholarlymarkdown.com/Scholarly-Markdown-Guide.html)
